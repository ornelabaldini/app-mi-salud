# app-mi-salud
# Registro de Pacientes y Medicamentos

Este proyecto es una aplicación de escritorio en Python que permite registrar y gestionar información médica de pacientes, incluyendo síntomas y medicaciones. La interfaz gráfica se construyó utilizando `CustomTkinter` y `Tkcalendar`, lo que permite a los usuarios interactuar de manera sencilla para ingresar y consultar registros médicos.
La aplicación está diseñada para registrar los siguientes datos de los pacientes:

- **Nombre y Edad** del paciente.
- **Síntomas**: Descripción del síntoma, fecha y hora.
- **Medicación**: Descripción de la medicación, dosis, fecha y hora.

Cada paciente puede tener varios registros de síntomas y medicaciones, y los usuarios pueden ver todos los registros de los pacientes o consultar los registros más recientes de un paciente específico.

## Requisitos

- Python 3.x
- CustomTkinter
- Tkcalendar
